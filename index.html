Primer desafío complementario

En este desafío se utilizará un bucle for para calcular el valor total de una compra de supermercado.

Modo de operación del algoritmo

  1. Mediante prompt se deberá ingresar el total de productos diferentes comprados
  2. Luego se deberá ingresar uno a uno el precio de cada producto y la cantidad
  3. Una vez ingresado todos los artículos se mostrará en la consola el precio total y la cantidad de IVA pagado
